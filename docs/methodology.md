# Methodology

## Goal
Build a VSAC-style, ingredient-level value set for ACE inhibitors, using
RxNorm as the primary code system and ATC as a cross-reference.

## Steps taken

1. **Identified the class.** ATC class `C09A` / `C09AA` = "ACE inhibitors, plain",
   confirmed via RxClass (`https://mor.nlm.nih.gov/RxClass/`).

2. **Set classification source to ATC.** RxClass supports multiple sources for
   the same class name (ATC, ATCPROD, VA, DailyMed, FDASPL, MEDRT). Selecting
   the wrong one silently changes the grain of the result:
   - `ATC` → ingredient-level (RxNorm TTY = `IN`) — what we want
   - `ATCPROD` → product-level (TTY = `SCD`/`SBD`/pack types) — too granular
   - `VA` (a different class tree entirely, e.g. class id `CV800`) → do not use;
     free-text search for "ACE INHIBITORS" can match this instead of the ATC tree

   Fix: search by exact class ID (`C09AA`) rather than free text, or navigate
   the tree directly to `ATC → C → C09 → C09A → C09AA`.

3. **Pulled ingredient list.** With source = ATC, the class page reported
   **14 RxNorm ingredients** in `C09AA`, each with `Type = IN` and `Relation =
   INDIRECT` (indirect because attached to child node `C09AA` while `C09A` was
   the node clicked — same data either way).

4. **Cross-checked against static ATC list.** The WHO ATC index defines
   `C09AA01`–`C09AA16` (16 chemical entries). RxNorm returned 14 — the gap is
   expected, since not every ATC-listed molecule has (or needs) its own
   US RxNorm ingredient concept, and some ATC entries map to combination
   or discontinued products instead.

5. **Corrected an earlier assumption.** Static/offline references (including
   an earlier draft of this project) listed benazepril's RxCUI as `19484` and
   assumed cilazapril/imidapril had no RxNorm ingredient concept at all. The
   live RxClass pull showed:
   - benazepril → `18867` (not 19484)
   - cilazapril → `21102` (does exist in RxNorm, contrary to the earlier assumption)
   - imidapril → `60245` (also exists)

   **Lesson:** always verify ingredient RXCUIs against the live API/browser
   before publishing — static tables (including ones generated by an LLM from
   training data) can be stale or wrong.

6. **Export.** Used RxClass's "Show source data" toggle to reveal the ATC code
   per row, then exported/copied the table into `data/ace_inhibitors_ingredients.csv`.
